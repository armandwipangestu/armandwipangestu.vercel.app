- Generating Public Key SSH

```shell
ssh-keygen -t rsa -b 4096 -C "your@email.com"
```

- Get PID

```shell
eval $(ssh-agent -s)
```

- Add Authentication Agent

```shell
ssh-add ~/.ssh/id_rsa
```

- Copy Public Key and Paste to GitHub Account

```shell
cat .ssh/id_rsa.pub
```
